# ADR-002: 初期リリースのプロトコルスコープ

## ステータス
**承認済み** (2026-06-20)

## コンテキスト
参考ツール（DummyPLC）は以下の4つのプロトコルをサポートしている:
1. 三菱 MCプロトコル / SLMP
2. OMRON FINS
3. KEYENCE 上位リンク / MCプロトコル
4. 横河 パソコンリンク

初期リリースでどこまでのスコープをカバーするか決定する必要がある。

## 決定
初期リリースでは **三菱MCプロトコル / SLMP のみ** に集中する。

## 理由
1. **ユーザー要求**: 「特に三菱PLCの機種やコマンドを充実させたい」と明示的に要求されている
2. **品質優先**: 1つのプロトコルを深く・正確に実装する方が、複数を浅く実装するより価値が高い
3. **マニュアル忠実性**: 三菱の公式マニュアルに忠実な実装を優先する
4. **拡張性**: プロトコルハンドラーをインターフェース化し、将来的に他メーカーのプロトコルを追加可能な設計とする

## 将来の拡張候補
- OMRON FINS（UDPポート9600）
- KEYENCE 上位リンク
- KEYENCE MCプロトコル
- 横河 パソコンリンク
- Modbus TCP/RTU
